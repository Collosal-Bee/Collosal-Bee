### 🚀 Featured Enterprise Projects

#### 1. [CommercePulse Analytics Pipeline]([https://github.com/Collosal-Bee/commercepulse-pipeline])
* **Scope:** Architected an event-driven ETL pipeline to unify highly irregular historical batch exports and high-velocity live e-commerce streams into a single analytics ecosystem.
* **Tech Stack:** Python (Pandas), MongoDB, Google Cloud BigQuery, SHA-256 Hashing.
* **Impact:** Engineered an in-memory reconciliation layer to dynamically resolve severe vendor schema drift and deduplicate payloads. Decoupled raw storage from compute to provide business teams with strictly-typed, high-performance BI tables.

#### 2. [SkyLogix Weather Data Pipeline]([https://github.com/Collosal-Bee/skylogix-weather-pipeline])
* **Scope:** Built an automated, end-to-end data pipeline to ingest real-time API weather data across major African logistics hubs.
* **Tech Stack:** Python, Apache Airflow, MongoDB (Staging), PostgreSQL (Data Warehouse).
* **Impact:** Automated the extraction, NoSQL staging, and relational transformation of JSON payloads via Airflow DAGs, ensuring reliable, highly-available structured data for downstream reporting.

#### 3. [Integrated Testing Platform]([https://github.com/Collosal-Bee/sap-quiz-app])
* **Scope:** Built a custom full-stack web engine to simulate complex enterprise relational logic for SAP Business Technology Platform (BTP).
* **Tech Stack:** Python, Streamlit, SQL, Cloud Hosting.
* **Impact:** Curated complex logic scenarios that helped 50+ users bridge knowledge gaps and successfully prepare for SAP Integration Suite and HANA certification exams.

#### 4. [Serverless Cloud Web Application]([https://github.com/Collosal-Bee/serverless-cloud-resume])
* **Scope:** Designed and deployed a highly available architecture decoupling frontend assets from backend compute.
* **Tech Stack:** AWS (Lambda, S3, API Gateway, DynamoDB), CI/CD workflows.
* **Impact:** Implemented automated deployment pipelines and integrated REST APIs, demonstrating secure cloud development while maintaining near-zero costs during idle times.
