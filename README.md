### 🚀 Featured Enterprise Projects

#### 1. [Pinnacle Quant Studio (WFO Engine)](https://github.com/Collosal-Bee/pinnacle-quant-viewer)
* **Scope:** Architected a decoupled machine learning pipeline to process massive compressed datasets, identifying mispriced markets and optimizing fractional Kelly criterion staking strategies.
* **Tech Stack:** Python, Optuna (Machine Learning), Streamlit, Plotly, Linux VPS.
* **Impact:** Engineered a Walk-Forward Optimization (WFO) backend to prevent historical overfitting. Developed a decoupled, high-performance Viewer App to visualize Monte Carlo stress tests and granular equity curves for verified out-of-sample data.

#### 2. [CommercePulse Analytics Pipeline](https://github.com/Collosal-Bee/commercepulse-pipeline)
* **Scope:** Architected an event-driven ETL pipeline to unify highly irregular historical batch exports and high-velocity live e-commerce streams into a single analytics ecosystem.
* **Tech Stack:** Python (Pandas), MongoDB, Google Cloud BigQuery, SHA-256 Hashing.
* **Impact:** Engineered an in-memory reconciliation layer to dynamically resolve severe vendor schema drift. Decoupled raw storage from compute to provide business teams with strictly-typed, high-performance BI tables.

#### 3. [SkyLogix Weather Data Pipeline](https://github.com/Collosal-Bee/skylogix-weather-pipeline)
* **Scope:** Built an automated, end-to-end data pipeline to ingest real-time API weather data across major African logistics hubs.
* **Tech Stack:** Python, Apache Airflow, MongoDB (Staging), PostgreSQL (Data Warehouse).
* **Impact:** Automated the extraction, NoSQL staging, and relational transformation of JSON payloads via Airflow DAGs, ensuring highly-available structured data for downstream climate analytics reporting.

#### 4. [Serverless ATS Resume Generator](https://github.com/Collosal-Bee/ats-resume-generator)
* **Scope:** Developed a serverless Streamlit application that dynamically parses raw Markdown text into strictly formatted, ATS-compliant PDF documents.
* **Tech Stack:** Python, Streamlit Cloud, WeasyPrint, CI/CD.
* **Impact:** Resolved complex Linux environment dependency conflicts within the deployment pipeline. Engineered custom CSS rule-sets and smart parsers to enforce optimal tagging structure, guaranteeing generated documents pass enterprise Applicant Tracking Systems.
